Discogs Integration Configuration

In order to download images from discogs and use some other features of sliMpd you have to:

  1. Create a Discogs account
  2. Register sliMpd as an application in your Discogs Account
  3. Add Consumer Key and Consumer Secret to your sliMpd configuration file
  4. Authorize sliMpd to use your Discogs account by clicking the link below
